Help

ILC is a transliteration app, it lets you type in English/Latin script and get text in your native Indian language script.

How to use

  1. Choose your language from the top menu bar.
  2. Type your text content in the first text box. Transliterated text in your chosen language script will appear in the next section.
  3. Indian script characters are mapped to English characters (which sound the same). Refer to the "Character Codes" table to learn about how these are mapped. The Character Code table also function as a keyboard.
